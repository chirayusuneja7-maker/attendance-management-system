# 📚 Attendance Management System

A full-stack **MERN Stack** web application that helps students manage attendance records securely. It features JWT authentication, attendance analytics, search & filtering, Excel export, and a responsive dashboard built with React and Bootstrap.

---

## 🚀 Features

✅ JWT Authentication (Login/Register)

✅ Protected Routes

✅ Add Attendance

✅ Edit Attendance

✅ Delete Attendance

✅ Search Attendance Records

✅ Filter by Status

✅ Dashboard Statistics

✅ Attendance Pie Chart

✅ Export to Excel

✅ Loading Spinner

✅ Toast Notifications

✅ SweetAlert2 Confirmation

✅ Responsive Bootstrap UI

---

## 🛠 Tech Stack

### Frontend

- React.js

- Axios
- Chart.js
- React Toastify
- SweetAlert2

### Backend

- Node.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/chirayusuneja7-maker/attendance-management-system.git
```

### Install Client

```bash
cd client
npm install
npm run dev
```

### Install Server

```bash
cd server
npm install
npm start
```
